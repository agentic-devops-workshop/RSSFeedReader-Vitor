# Quickstart: MVP RSS reader

## Prerequisites
- .NET 6 SDK or newer

## Setup
1. Clone the repository
2. Navigate to `backend/` and run:
   ```sh
   dotnet run
   ```
3. Navigate to `frontend/` and run:
   ```sh
   dotnet run
   ```

## Usage
- Open the frontend app in your browser (URL provided by Blazor dev server)
- Add RSS/Atom feed URLs using the input field
- View your subscription list in the UI

## Notes
- No data is persisted; subscriptions are lost on restart
- No feed fetching or parsing in MVP
